# ChillWithYou — 한글 패치 설치 방법 (간단 요약)

중요 요약
- 릴리즈 ZIP(배포용 패키지)을 사용하면: 압축을 풀어서 나온 폴더/파일을 게임이 설치된 폴더(게임 실행 파일(.exe) 있는 곳)에 그대로 복사(병합)하면 됩니다. 별도 작업 불필요.
- 반면 이 GitHub 레포(소스/리소스 저장소)를 직접 clone/pull 해서 사용하면: 레포에는 BepInEx 핵심 파일들이 일부러 제외되어 있습니다. 따라서 레포를 가져온 뒤에는 사용자가 별도로 BepInEx를 다운로드하고 설치해야 합니다.

압축(릴리즈 ZIP)으로 설치할 때
1. 게임 폴더 백업(권장).
2. 릴리즈 ZIP의 압축을 풉니다.
3. 압축에서 나온 항목들(예: BepInEx/ 폴더, plugins/, Translation/, ttf 등)을 게임 설치 폴더에 그대로 복사(병합)합니다.
   - 예: 압축 안의 `BepInEx` 폴더는 게임 폴더 안에 `BepInEx`로 병합됩니다.
4. 기존 파일 덮어쓰기 메시지가 나오면 백업 후 덮어쓰세요.
5. 게임 실행 후 `BepInEx/LogOutput.log` 또는 `BepInEx/logs/`에서 플러그인 로드 로그를 확인하세요.

레포를 clone/pull 해서 사용하는 경우 (주의)
- 이 저장소는 BepInEx 코어(대용량 .dll 파일들 등)를 모두 포함하지 않습니다. 이유: 불필요한 바이너리 중복/라이선스/업데이트 문제를 방지하기 위해 일부 BepInEx 파일은 배포용 ZIP에만 포함하거나 아예 제외했습니다.
- 따라서 레포를 그대로 복사해서 게임 폴더에 넣는 것만으론 작동하지 않을 수 있습니다. 이 경우 다음을 반드시 수행하세요:

레포 사용 후 BepInEx 설치 방법
1. BepInEx 최신 릴리즈 페이지로 이동:
   - https://github.com/BepInEx/BepInEx/releases
2. 게임과 Unity 버전에 맞는 BepInEx를 다운로드하고, 공식 설치 가이드를 따라 게임 폴더에 설치합니다.
3. 그 다음 이 레포(또는 릴리즈에서 압축 해제한 파일)의 `BepInEx/plugins/` 및 `BepInEx/Translation/` 등의 폴더를 기존 게임 폴더에 복사하여 적용합니다.

한줄로 정리하면
- 릴리즈 ZIP: 압축 풀고 게임 폴더에 넣으면 끝.
- Git 레포(또는 clone 방식): BepInEx 핵심 파일 일부가 레포에 없으니, 레포를 가져온 뒤 별도 BepInEx 설치가 필요합니다. (링크에서 다운로드 후 설치)

문제 발생 시 확인할 것
- BepInEx가 설치되어 있는가?
- `BepInEx/plugins`에 플러그인 DLL이 있는가?
- 번역 파일(.txt/.json)이 올바른 경로에 있는가?
- `BepInEx/LogOutput.log`에 에러가 있는가?

필요하면 릴리즈용 ZIP에 어떤 파일을 포함시킬지(예: 플러그인만 포함 / BepInEx 전체 포함 여부) 추천하거나, "레포에서 clone 한 뒤 자동으로 BepInEx를 설치하도록 돕는 스크립트"도 만들어 줄게.
