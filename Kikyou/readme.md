```
# Kikyou Mod

캐릭터를 키쿄로 바꾸는 모드입니다.

간단히:
- 키쿄 모델을 적용해 게임 캐릭터를 대체합니다.
- 모델링이 없으면 동작하지 않습니다.

상태:
- 개발 중입니다.
- 현재 프로젝트에는 키쿄 모델 파일이 포함되어 있지 않습니다.

중요:
- 키쿄 모델은 부스(유료)에서 구입한 모델입니다. 모델 파일은 재배포가 금지되어 있어 이 저장소에는 포함되지 않습니다.
- 즉, 이 저장소를 내려받아도 모델이 없으면 사용할 수 없습니다.

설치(요약):
- 모델을 별도로 구입하여 적절한 위치에 배치해야 동작합니다.
- 자세한 설치법과 파일 구조는 향후 문서로 정리할 예정입니다.

면책:
- 모델 라이선스는 구매처(부스)의 규정을 따르세요.
- 이 저장소는 코드 보관 및 개발용이며, 모델 배포를 목적으로 하지 않습니다.
```
