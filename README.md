# ChillWithYou-Mods

A collection of BepInEx mods for Chill With You


## Mods

### YtDownloader

<img width="513" height="555" alt="스크린샷 2025-12-01 041530" src="https://github.com/user-attachments/assets/2488cc9d-177a-48ce-b82e-cc81c083d3d2" />



YouTube에서 음악을 다운받아 게임에 추가할 수 있는 모드

**기능**
- YouTube URL로 mp3 다운로드
- 여러 곡 동시 다운로드 (URL 줄바꿈으로 구분)
- 다운로드 진행률 표시
- 다운로드 취소

**단축키**
- `F7` - 다운로드 패널 열기/닫기

---

## 설치 방법

1. [Releases](https://github.com/Ich-mag-dich/ChillWithYou-Mods/releases)에서 zip 다운로드

2. 압축 해제 후 게임 폴더에 복사:
   - `BepInEx/` (폴더)
   - `. doorstop_version`
   - `doorstop_config.ini`
   - `winhttp.dll`

3. 게임 폴더 예시:
   ```
   C:\Program Files (x86)\Steam\steamapps\common\Chill With You\
   ```

4. 게임 실행 후 `F7`
